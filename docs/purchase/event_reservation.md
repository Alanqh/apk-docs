# 预约活动

游戏可以通过【预约活动】更大限度的扩散游戏的预约量和福利信息，吸引泛用户的关注，提升预约人数，从而进一步提高游戏首发登录转化。

> 目前支持【首发新游】和老游的【大版本更新】
