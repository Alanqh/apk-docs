# 代金券

代金券活动能更好的促进玩家付费，一般可以设置“满6-3”等小额付费，从而提高付费率。目前233乐园只做共担代金券。

若同意做代金券活动，需要将活动内容发送邮件至233运营部门和商务的邮箱，同时准备一张宣传图，用于代金券活动的banner资源位展示，尺寸720*360，不大于200k。
