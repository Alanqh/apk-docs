# 如何申请活动

## **一、预约活动&签到活动&综合活动**

因目前开发者平台功能的限制，所以需要提前将素材、资讯打包发给对接运营。

新游在开始对接时即可沟通、准备需求素材+活动物料，版本预约素材需尽早发送，其他活动可在游戏上线前2天发送

老游最晚在版本开启前半个月与对接运营沟通，最迟在版更前一天将素材发送（其中版本预约活动在版更前2周即可开启，需提前申请）

申请时需要将信息注明，需要包括以下内容：

游戏名称

申请原因

需求时间

游戏活动内容

具体奖励及奖励价值
